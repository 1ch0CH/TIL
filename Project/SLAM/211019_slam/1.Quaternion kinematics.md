# 1. Quaternion definition and properties
## 1-1. Definition of quaternion
- Quaternion definition 

$\Q = a + bi + cj + dk ∈ H$\

where {a, b, c, d} ∈ R, and {i, j, k} are three imaginary unit numbers defined so that
$$ 
i^2 = j^2 = k^2 = ijk = −1 
$$


$$ij = −ji = k , jk = −kj = i , ki = −ik = j .$$
